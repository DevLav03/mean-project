# MEAN Stack Application 

This is a simple full-stack project built using MEAN stack.

## Technologies Used
- Angular
- Node.js
- Express.js
- MongoDB

## Features
- User Login & Register
- JWT Authentication
- REST API
- MongoDB Database
- Angular Standalone Architecture

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
ng serve
